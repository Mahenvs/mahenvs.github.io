Privacy Policy

1. Data Collection

This extension collects user-provided details such as name, email, job title, city, postal code, and company name. The data is stored locally on the user's device and is not transmitted to any external servers.

2. Data Usage

The collected information is used solely to pre-fill job application forms on Indeed, making the application process faster and more efficient.

3. Data Sharing

We do not share, sell, or transmit any user data to third parties.

4. Data Storage & Security

All user data is stored locally in Chrome storage. Users can delete their data at any time by removing the extension or clearing extension storage.

5. Permissions

This extension requires scripting permissions to detect job application forms and autofill fields. It does not submit applications automatically.
